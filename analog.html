Arduino-ի անալոգային և թվային ելքեր

Այս դասի ընթացքում կհասկանաք, թե ինչով է տարբերվում թվային ազդանշանը անալոգայինից։ Միացնենք 2 լեդ և կարգավորենք դրանց պայծառությունը։ Քննարկենք տվյալի int տիպը, պայմանի if օպերատորը և analogWrite հրամանը։

Անալոգային ազդանշանը ժամանակի ընթացքում անընդհատ փոփոխվում է։ Բնության մեյ բոլոր ինֆորմացիաները անալոգային են՝ գծովի ալիքները, նվագալարի տատանումները և այլն։ Ի սկզբանե մարդիկ ձայնագրում էին ինֆորմացիան (ձայներ, նկարներ, տեսանյութեր) անալոգային սարքերի օգնությամբ։ Սակայն անալոգային ազդանշանները զգայուն են աղմուկի և արգելքի նկատմամբ

Թվային ազդանշանը հաղորդվում է 1 և 0 տեսքով, համակարգիչների և թվային տեղնիկայի համար դա հեշտ է իրականացնել(ազդանշանի կա կամ չկա)։ Համակարգիչների օպերատիվ հիշողության համար կիրառում են կոնդեսատորներ, 1 լիցքավորված կոնդեսատորը 1բիթ է։

Համակարգիչների ի հայտ գալուց հետո անալոգային ազդանշանները սկսեցին փոխակերպել թվայինի, քանի որ անալոգային ազդանշանը ձայնագրման ժամանակ ենթակա է խեղաթյուրման և փոխանցման թուլացման։

Arduino-ի անալոգային և թվային ելքեր

Եթե անհրաժեշտ է կարգավորել ելքային լարումը, անհրաժեշտ է կիրառել այն պիները, որոնք նշված են «~» ցիմվոլով։ Arduino Uno-ի դեպքում՝ 3, 5, 6, 9, 10, 11։ Անալոգային պորտերի օգնությամբ հնարավոր է հաղորդել ցանկացած լարում՝ 0-5վարկյան, իսկ թվային ելքերը հնարավոր է միայն անջատել և միացնել։ Անալոգային պորտերը կիրառում են ԼԻՄ (լայնաիմպուլսային մոդուլացիա), որի օգնությամբ նմանագործում է անալոգային ազդանշանը։

Թվային և անալոգային ազդանշանների տարբերությունը հասկանալու համար։ Միացնում են Arduino տպասալին լեդ լույս, սակայն այս անգամ անալոգային ելքին՝ ~9։ Պորտ 9-ը հնարավոր է աշղատի որպես թվային ելք։ Սակայն, եթե digitalWrite ֆունկցիան փոխենք analogWrite ֆունկցիայով, ապա HIGH և LOW արժեքների փոխարեն կարող ենք տեղադրել ցանկացած թիվ 0-255 միջակայքից։ 

                       int svet = 0; // լեդի սկզբնական պայծառությունը
                       int fade = 5; // լեդի պայծառության փոփոխման քայլը
           
                       void setup() {
                           pinMode(9, OUTPUT); // используем Pin9 для операции вывода
                       }
           
                       void loop() {
           
                           analogWrite(9, svet);
                           svet = svet + fade;
                           if (svet == 0 || svet == 255) {
                               fade = -fade;
                       }
           
                       delay(20); 
                       }

Կոդի մեկնաբանություն

  • analogWrite (pin,value) ֆունկցիա -pin` ելքի պորտ, որին հաղորդվում է ազդանշան, value` 0-255 միջակայքից արժեք, կիրառվում է լեդի պայծառությունը ղեկավարելու համար կամ էլեկտրաշարժիչի արաքգությունը փոփոխելու համար․
  • svet փոփոխականը ունի սկզբնական 0 արժեքը, ցիկլի յուրաքանչյուր շրջափուլի ժամանակ արժեքը մեծանում է տրվա fade արժեքով։
