Arduino-ում if... else տրամաբանական օպերատոր

Arduino-ում if կոնստրուկցիան ստուգում է պայմանը, եթե պայմանը ճիշտ է, ապա կատարվում է ձևավոր փակագծերի մեջ գրված կոդի հատվածը: Առանց ձևավոր փակագծերի կատարվում է միայն կոդի մեկ տողը, որը հաջորդում է պայմանի օպերատորին: Խորհուրդ է տրվում մշտապես կիրառել ձևավոր փակագծեր, քանի որ այդ պարագայում ծրագրում սխալներ թույլ չի տրվելու:

Arduino-ում if...else պայմանի նկարագրություն

Arduino-ում if-ն օգտագործվում է համեմատական ​​օպերատորների հետ՝ ստուգելու, թե արդյոք պայմանները ճշմարիտ են: Ձևավոր փակագծերը տրամաբանորեն խմբավորում են բազմաթիվ տողեր մեկ բլոկի մեջ: Ներքոհիշյալ օրինակում ստուգվում է, արդյոք water փոփոխականի արժեքը մեծ է տրված թվից. 

            if (water > 100) { digitalWrite (12, HIGH ); digitalWrite (10, LOW ); }

            if (water < 100) { digitalWrite (12, LOW ); digitalWrite (10, HIGH ); }
If..else կոնստրուկցիան մեծ վերահսկողություն է ապահովում ծրագրի վրա, քան մեկ if օպերատորը: Եվ դա թույլ է տալիս սահմանել գործողություններ ոչ միայն այն դեպքի համար, երբ արտահայտությունը ճիշտ է, այլ նաև հակառակ դեպքի համար (երբ արտահայտության արժեքը false է). 
            if (water > 100) { digitalWrite(12, HIGH); digitalWrite(10, LOW); }

            else { digitalWrite(12, LOW); digitalWrite(10, HIGH); }
Arduino IDE-ում else-ին կարող է հաջորդել մեկ այլ if՝ ստեղծելով ստուգումների մի ամբողջ շղթա: Պայմանների ստուգումները կկատարվեն մեկը մյուսի հետևից, մինչև չհանդիպեն իրական արտահայտության: Օրինակ 
            if (water > 100) { digitalWrite (12, HIGH ); digitalWrite (10, LOW ); }

            else if (water > 200) { digitalWrite (12, LOW ); digitalWrite (10, HIGH ); }

Հավասարության օպերատորներ, պայմանների հերթականություն

        x == y (x-ը հավասար է y-ին)
        x != y (x-ը հավասար չէ  y-ին)
        x < y (x-ը փոքր է  y-ից)
        x > y (x-ը մեծ է y-ից)
        x <= y (x-ը փոքր է կամ հավասար y-ին)
        x >= y (x-ը մեծ է կամ հավասար  y-ին)
