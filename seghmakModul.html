Ինչպես միացնել սեղմակով մոդուլը Arduino-ին

KY-004-ը մոդուլ է Arduino Nano միկրոկառավարիչի համար: Մոդուլն ունի սովորաբար բաց կոնտակտներ, այսինքն՝ կոնտակտները փակվում են, երբ կոճակը սեղմվում է, և սենսորի ելքում հայտնվում է ցածր մակարդակի լարում: է «S» ելքային կոնտակտի և Arduino-ի «5V» պինի միջև եռակցված 10 կՕմ դիմադրություն:

Մոդուլը կարող է միացված լինել ինչպես ընդհանուր նշանակության թվային պինին, այնպես էլ անալոգային սենսորների մուտքային պինին: KY-004-ը թույլ է տալիս կոճակը միացնել Arduino-ին առանց breadboard օգտագործելու, հակառակ դեպքում ոչ մի տարբերություն չկա սովորական տակտային կոճակը միկրոկառավարիչին միացնելուց:

 boolean buttonWasUp = true;
  boolean buttonIsUp  = false;
   boolean ledEnabled  = false;
    
    void  setup() {
      pinMode(13, OUTPUT);
      pinMode(A1, INPUT_PULLUP);
    }
    
     void  loop() {
      buttonIsUp = digitalRead(A1);
     if (buttonWasUp && !buttonIsUp) {
        delay(10);
        buttonIsUp = digitalRead(A1);
    
        if (!buttonIsUp) {
          ledEnabled = !ledEnabled;
          digitalWrite(13, ledEnabled);
        }
      }
    
      
      buttonWasUp = buttonIsUp;
    }

Կոդի բացատրություն

1. կոճակի մուտքագրումը (pin A1) կազմաձևված է որպես INPUT_PULLUP. Երբ կոճակը սեղմված է, մուտքագրումը կլինի տրամաբանական «զրոյական» (false) արժեքը, իսկ buttonIsUp փոփոխականում արժեքը կլինի «true» , երբ կոճակը սեղմված չի:

2. digitalWrite-ին չի տրվում HIGH կամ LOW արժեքը, այլ օգտագործվում է ledEnabled փոփոխականը: Կախված նրանից, թե որ արժեքն է հաշվարկվել false կամ true, լուսադիոդը կմիանա կամ կանջատվի:
