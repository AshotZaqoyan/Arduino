switch...case օպերատոր

Switch օպերատորը նման է if օպերատորին, այն վերահսկում է ծրագիրը՝ նշելով այլընտրանքային կոդը, որը պետք է կատարվի տարբեր պայմաններում: Եթե ​ անհրաժեշտ է ընտրել երկուսից ավել տարբերակներից, ապա կարող ենք մի քանի անգամ օգտագործել if ... else if կամ switch ... case հայտարարությունը:
Arduino-ում switch case օպերատորը թույլ է տալիս սկետչում ստեղծել հարմար կոնստրուկցիա,կախված փոփոխականի արժեքից թույլ է տալիս ծրագրում գործողությունների ճյուղավորում կատարել: default (լռելյայն) օպերատորի առկայությունը պարտադիր չէ, իսկ break (ընդհատել) օպերատորի առկայությունը պարտադիր է: If… else պայմանի օպերատորի և switch...case ընտրության օպերատորի օգնությամբ կառուցվում է ողջ ծրագրի տրամաբանությունը:
Switch օպերատորը համեմատում է փոփոխականի արժեքը case օպերատորի արժեքների հետ: Երբ գտնում է case, որի արժեքը հավասար է փոփոխականի արժեքին, ապա կատարվում է ծրագրի այդ բլոկը: Օրինակ. 

        int data = 2;

        switch(data){
            case 1: Serial.println("1");break;
        
            case 2: Serial.println("2");break;
        
            case 3: Serial.println("3");break;
        }
break-ը Arduino IDE-ում օգտագործվում է հաջորդ պայմանին անցնելու համար, եթե նախորդ պայմանը ճիշտ չէր: Arduino default օգտագործվում է կոդը գործարկելու համար, եթե switch-ի բոլոր պայմանները ճիշտ չեն: 
            int data = 5;

switch(data){
    case 1: Serial.println("1"); break;

    case 2: Serial.println("2"); break;

    case 3: Serial.println("3"); break;

    default: Serial.println("___"); break;
}
Ծրագրավորման բոլոր լեզուներն ունեն մի շարք հսկիչ հրամաններ, որոնք ապահովում են նույն կոդի բազմակի կատարում (առանց ցիկլով), համապատասխան կոդի ընտրություն (ifպայման) և ընթացիկ կոդից դուրս գալու հրահանգներ: Arduino IDE ծրագրավորման լեզուն C/C++ լեզվից փոխառել է պահանջվող հսկիչ հրամանների մեծ մասը, ուստի դրանց սինտաքսը հիմնականում նույն է:
